# Expense Report（支出统计）一页说明

这是一个本地记账 Skill。
你只要像聊天一样发一句话，它就能记账、分类、汇总，并按日/周/月/年出报告。

## 你能得到什么

- 随手记账（不用打开复杂 App）
- 多币种支持（美元、欧元、韩元等）
- 自动换算成人民币出报告
- 遇到“待分类”会给你 3 个建议
- 你确认一次后会学习，下次自动归类
- 数据全部本地保存，不怕服务停更

## 你平时怎么用

直接发：

- `午饭 32元`
- `咖啡 USD 4.5`
- `补录 3月1日 打车 45`

如果系统说“待分类”，你只要回一个分类就行。

## 报告会给你什么

- 总消费（人民币）
- 分类占比
- 大额支出提醒
- 和上一周期对比（涨了还是降了）

## 数据放在哪里

```text
shared/expense-report/
├── config.json        # 提醒时间、分类、投递设置
├── entries.jsonl      # 每笔消费
├── fx-rates.json      # 汇率缓存
└── reports/           # 日/周/月/年报告
```

## 一句话总结

这是一个“越用越懂你”的本地记账助手：
输入简单、报告清晰、数据可控。
